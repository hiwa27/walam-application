# Build Unsigned IPA

Run this on a macOS machine (local Mac or cloud Mac runner):

```bash
cd /path/to/walam\ aplication
chmod +x scripts/build_unsigned_ipa.sh
./scripts/build_unsigned_ipa.sh
```

Output file:

`build/ios/unsigned/walam_app_unsigned.ipa`

This IPA is unsigned and ready for your signing flow.
